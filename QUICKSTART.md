# Быстрый старт

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. На Windows установите **Npcap** (для scapy):
   - Скачайте с https://npcap.com/
   - Установите с опцией "WinPcap API-compatible Mode"

## Запуск

### Базовый запуск (автоопределение сети):
```bash
python main.py --auto
```

### С указанием сети:
```bash
python main.py --range 192.168.1.0/24
```

### С выбором формата:
```bash
python main.py --auto --format csv --output my_scan
```
Можно указывать не только имя формата, но и цифру:
- `1` — JSON (по умолчанию)
- `2` — CSV
- `3` — TXT  
Если параметр `--format` опустить полностью, приложение попросит выбрать формат при запуске.

## Важно

- Запускайте от имени **администратора** для полного функционала
- Первый запуск может занять время (установка зависимостей)
- Сканирование сети /24 занимает примерно 1-2 минуты

## Примеры вывода

После сканирования будут созданы файлы:
- `network_scan.json` - структурированные данные
- `network_scan.csv` - табличный формат
- `network_scan.txt` - читаемый отчет
